## If statement

***C++***

```
==, !=, !, &&, ||, ()
```

```cpp
int var = 1;

if (var == 1) {
    ...
} else if (var == 2) {
    ...
} else {
    ...
}
```

***Python***

```
==, !=, not, and, or, ()
```

```python
var = 1

if var == 1:
    ...
elif var == 2:
    ...
else:
    ...
```
